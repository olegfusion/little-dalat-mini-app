# Session — July 20, 2026

## Fly-to-Cart Animation

New component `FlyToCart.tsx`:
- Pure DOM + Web Animations API (no extra dependencies)
- **From list**: image enlarges to ×5 at center of screen, then flies to cart icon in an arc
- **From modal**: direct arc to cart (no pause)
- Duration: 1350ms total, `cubic-bezier(0.22, 1, 0.36, 1)`
- Cart icon bounces on arrival (`cart-bounce` keyframe in `index.css`)
- Only triggers on actual add-to-cart (not when opening variant picker)
